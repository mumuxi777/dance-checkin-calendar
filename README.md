# 打卡日历

一个可以自定义标题、打卡类型和颜色的月历打卡工具。纯静态网页，无需后端，数据保存在浏览器本地（`localStorage`）。

## 使用

- 打开网页后点击任意日期，选择当天的打卡类型
- 点标题 / 副标题可以直接改名（比如改成"健身打卡"之类的）
- 点类型标签（Jazz / Hip-Hop 这些）可以改名、换颜色、删除；点"+ 添加"可以新增类型
- 卡片右上角的调色板图标可以切换整体配色
- 底部可以导出/导入 JSON 备份

支持作为 PWA 安装：
- **Windows（Chrome / Edge）**：打开网页后点地址栏的"安装"图标，或菜单里的"安装应用"。
- **Android（Chrome）**：打开网页后点菜单里的"添加到主屏幕" / "安装应用"。

安装后会有独立的应用图标和窗口，支持离线打开。

## 数据说明

打卡数据只存在当前浏览器本地，不会上传到任何服务器，也不会在设备之间自动同步。清除浏览器数据或更换设备会导致记录丢失，建议定期用页面上的"导出备份"功能保存一份 JSON 文件。

## 开发

纯 HTML/CSS/JS，没有构建步骤。改完 `index.html` 直接推送到 `main` 分支，GitHub Pages 会自动重新发布。
